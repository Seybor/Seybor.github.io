<ul>
<li>Number.isFinite() - Определяет, является ли переданное значение конечным числом и возвращает логическое значение</li>
<li>Number.isInteger() - Определяет, является ли передаваемое значение целым числом и возвращает логическое значение</li>
<li>Number.isNaN() - Определяет, является ли передаваемое значение NaN и возвращает логическое значение</li>
<li>Number.parseInt() / parseInt() - Метод принимает в качестве аргумента строковое значение и возвращает целое число, которое соответствует указанному основанию системы счисления</li>
<li>Number.parseFloat() / parseFloat() - Метод принимает в качестве аргумента строковое значение и возвращает десятичное число(число с плавающей точкой)</li>
<li>toFixed() - Метод форматирует числовое значение в строковое, сохраняя при этом указанное количество десятичных знаков</li>
<li>toString() - Возвращает строковое представление переданного числа.В аргументе можно указать систему счисления</li>
</ul>
